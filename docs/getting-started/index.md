## Prerequisites

### For Python Users

- Python 3.12 or later
- uv (or pip) package manager

### For Node.js Users

- Node.js 20 or later
- npm package manager

### For Go Users

- Go 1.22 or later
- C compiler (for CGO)

### For C# Users

- .NET 8 SDK or later

### For Dart Users

- Dart SDK 3.0 or later

### For Rust Users

- Rust 1.91.1 or later
- Cargo package manager
